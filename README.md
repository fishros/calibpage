# 中文网提交翻译内容网页

便于用户对中文网的翻译结果进行矫正

# 运行环境

- Node.js 版本 14.17.0 以上
- 全局 Yarn 版本保持最新
- 现代浏览器和 IE11 以上

# 本地开发

1. 进入项目根目录，执行 `yarn` 命令安装依赖包；

3. 执行 `yarn dev` 命令启动项目；
